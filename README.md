📚 Library Management System

A responsive Library Management System built using HTML, CSS, and JavaScript. This project helps manage books, students, book issuing, and returns using the browser's Local Storage.

✨ Features

- 🔐 User Login Page
- 📊 Responsive Dashboard
- 📚 Book Management
- 👨‍🎓 Student Management
- 📖 Issue Books
- 🔄 Return Books
- 💾 Local Storage Support
- 📈 Charts & Statistics
- 🚪 Logout Functionality
- 📱 Mobile-Friendly Responsive Design

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- Chart.js

🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open "login.html" (or "index.html") in your web browser.
4. Start managing books and students.
